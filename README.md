# Khoa Dev - Universal Obfuscator V3.0

## Khoa Dev là Admin

Khoa Dev là Admin và là người phát triển công cụ **Universal Obfuscator V3.0**.

Đây là hệ thống đóng gói và làm rối mã nguồn đa ngôn ngữ, hỗ trợ bảo vệ code bằng cách biến đổi mã nguồn thành dạng khó đọc hơn nhưng vẫn có thể thực thi.

---

# Có thể làm gì

Universal Obfuscator V3.0 có thể:

- Làm rối mã nguồn Python
- Làm rối mã nguồn Lua / Luau
- Làm rối mã nguồn JavaScript
- Mã hóa nội dung source code
- Tạo biến ngẫu nhiên gây khó đọc
- Thêm đoạn mã rác (junk code)
- Đóng gói payload thành file mới
- Hỗ trợ nhập code trực tiếp hoặc đọc từ file
- Xuất file đã obfuscate

## Lua / Luau

Hệ thống:

- Mã hóa byte UTF-8
- Tạo bảng dữ liệu
- Giải mã bằng string.char
- Chạy bằng loadstring/load

## JavaScript

Hệ thống:

- Chuyển ký tự thành mã số
- Giải mã bằng String.fromCharCode
- Thực thi bằng Function()
